# twitter-trends-api
Twitter Trends API Exampes

## How to run
Please set your Twitter api keys env

```bash
export TWITTER_API_KEY='**************' \
export TWITTER_API_SECRET='**************' \
export TWITTER_ACCESS_TOKEN='**************' \
export TWITTER_ACCESS_TOKEN_SECRET='**************'
```

You can then run the api
```
node api
```
